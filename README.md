This project analyses manufacturing defect data to identify key cost drivers, defect patterns, and quality risks across products, inspection methods and defect severity.

The dashboard is designed to support quality and operations decision-making by focusing on the financial impact of defects rather than only defect volume.

The main goal is to help stakeholders quickly understand:

where defects occur,

which defects are most expensive,

and how defect trends evolve over time.
